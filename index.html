Welcome to IconKit.

IconKit is a dynamic icon generation library.You can create dynamic icon by code.

Example

Search *search =[[Search alloc] initWithSize:CGSizeMake(200, 200)];

search.fillColor =[UIColor redColor];

UIImage *image =search.image;
